# Electric-Car-Data-Analysis (전기차 데이터 분석)
## 프로젝트 개요
### 목표
- 전기차와 전기차 충전소 현황을 분석하여 현상황과 문제점 그리고 앞으로의 미래 도출
### 사용 스킬
- Jupyter Notebook (anaconda3)
### 분석 환경
- Windows 10 Education
### 참여 인원
- 2명
### 역할
- 송현준(팀장) : 프로젝트 총괄 및 발표
- 진병욱(나) : 필요한 데이터 조사 및 PPT 제작
- 데이터 분석은 공동

물론 딥하게 들어간건 아니지만 데이터 분석이라는 분야도 굉장히 매력적인 것 같다.
